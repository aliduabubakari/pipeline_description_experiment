Pipeline Summary:
- Pipeline ID: multilingual_product_review_pipeline
- Name: multilingual_product_review_analysis_v1
- Purpose: Enriches product reviews with language detection and sentiment analysis
- Execution model: batch
- Topology pattern: linear
- Business domain: data_science
- Notes: Sequential data pipeline analyzing multilingual product reviews. It performs language verification, LLM-based sentiment analysis, and feature extraction using containerized services.

Data Artifacts / I-O Identifiers:
- file: reviews.csv (Input product reviews)
- file: table_data_2.json (Formatted review data)
- file: lang_detected_2.json (Reviews with language detection)
- file: sentiment_analyzed_2.json (Reviews with sentiment scores)
- file: column_extended_2.json (Reviews with extracted features)
- file: enriched_data_2.csv (Final enriched output)

External Systems:
- filesystem: local_data_directory (/tmp/reviews/data)
- cloud_service: Docker Registry

Pipeline Steps:
- analyze_sentiment — analyze_sentiment
  - Objective: Perform sentiment analysis on review text
  - Mechanism: container_run
  - Inputs: file:lang_detected_2.json
  - Outputs: file:sentiment_analyzed_2.json
  - Step parameters:
    - input_file=lang_detected_2.json
    - output_file=sentiment_analyzed_2.json
    - model_name=distilbert-base-uncased-finetuned-sst-2-english
    - text_column=review_text
    - output_column=sentiment_score
  - Failure handling:
    - retries: 1
- detect_language — detect_language
  - Objective: Detect language of review text
  - Mechanism: container_run
  - Inputs: file:table_data_2.json
  - Outputs: file:lang_detected_2.json
  - Step parameters:
    - input_file=table_data_2.json
    - output_file=lang_detected_2.json
    - text_column=review_text
    - lang_code_column=language_code
  - Failure handling:
    - retries: 1

    - retries: 1

  - Mechanism: container_run
